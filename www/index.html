About the Drone Project for Embedded Linux - Spring 2015

For this project, our group was given a 3DR Hexacopter drone that will be programmed to fly to a fixed set of way stations and return to a base station using the program, Mission Planner. Upon reaching each way station the drone will take a series of images. These images are captured by implementing a Raspberry Pi that runs python scripts which communicate with a GPS module that activate the camera when a way station has been reached. When the drone returns to the base station, the images that were captured and saved to the database are transmitted to a central location. A web interface will display the images to create a composite view of the area covered by the drone. Consider that when the drone is at a given position, you must be able to rotate the camera at least a few different directions so as to capture a wider swath of area, rather than a single shot in a fixed direction.
